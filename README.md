# Twitch Ads Blocker Frontend

Frontend for the Twitch Ads Blocker dashboard.

## Tech Stack
- React
- TypeScript
- Vite

## Features
- Twitch OAuth login
- Browse categories
- View live streams
- Followed channels
- Ads blocking integration

## Development
    Install dependencies:
      bun install
    Run dev server:
      bun run dev
    Build project:
      bun run build
